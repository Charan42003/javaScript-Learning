Modules

Export

        syntax:
            export variables/functions/etc.. 
            (for more than one export)
            export {variable1, valriable2, funciton1, function2, etc, etc}

Import

        syntax:
            import {value} from 'path'
              (or)
            import value from 'path'
            (for more than one import from the same file)
            import {value1, value2, valuen} from 'path'

            (for re-naming and using an imported value)
            import value as newName _to_call from 'path'

            (to import all the exported values from a single file)
            import * as name from 'path'
            //calling the value
            name.variable;
            name.function();

            (to import and use default function)
            import {default as name} from 'path'
            name();
