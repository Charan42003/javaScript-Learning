Arrow Funciton

when we use arrow function. The function should be stored inside a variable

        syntax :

            let function_Name = (parameters) => {
                //code
            }

        example :

            let sum = (a, b) => {
                let s = a+b;
                return s;
            }
            sum(4, 5);

        If the code inside the function is of only one line and the line is returning a value
            let sum = (a, b) => a+b;
            sum(4,5)

        for single parameter (only if the number of parameter is one)
            let double = a => a*2;
            double(a);
