Rest parameter (...parameter_name)

rest parameter is used when the number of paramaters that has to be passed to the function is unknown
the parameters are stored in form of an array. 

        syntax: function function_name(...parameter_name)

        example: function sum(...a) {
            console.log(a);
            let res = 0;
            for(let i = 0; i < a.length; i++)
            {
                res += a[i];
            }
            console.log(res);
        }
        sum(4,5,6,2,3);
