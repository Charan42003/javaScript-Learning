for...of loop

used to iterate (go/mainpulate) to all the array/string elements without knowing the array length 

        syntax: 
            for(let variable_name of array_name) {
                //code
            }

        example: 
            let array = [54, true, "sct", 55, 4]
            for(let a of array) {
                console.log(a);
            }
