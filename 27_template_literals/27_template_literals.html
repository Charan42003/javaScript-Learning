Template Literals

this will allow us to use dynamic changing value stored in a variable inside a String
=> backticks will be used insted of single or double inverted commas 

        syntax :   ` normal string ${variable_name/ expressions}`

        example:
            let namee = "charan"
            let place = "bangalore"
            let age = 19
            let a = `Hello, My name is ${namee} and I am ${age} years old and I stay in ${place}`
            console.log(a)
