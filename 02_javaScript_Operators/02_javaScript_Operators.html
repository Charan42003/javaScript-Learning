JavaScript Operators

= : Assignes value

== : Checks whether left is equal to right

+ : Used to perform Addition

- : Used to perform subtraction

/ : Used to perform divison

* : Used to perform Multiplication

% : Used to get the Remainder

+= , -=, /=, *= : Used to perform x = x + 1 etc....

** : Used to get the power


Increment and Decrement Operator

++ : Used to increase the value by one(1)

-- : Used to decrease the value by one(1)

typeof Operator

This operator is used to know the type of the operator
syntax: typeof"value"

Template Literal

Used to print the value stored in a variable
syntax: `${variable name}`

Comparison Operator

used to compare two values

like
== : equal to //datatype is not seen
=== : equal value & equal type
!= : not equal
!== : not equal value or not equal type

> : greater than
< :less than
<= : less than or equal to
>= : greater than or equal to

Logical Operator

Used to carry out logical tests

=> && : and, example: 5>3 && 4>1 //flase
=> || : or, example: 4>5 || 4>2 //true
=> ! : not, example: !(true) //fales
