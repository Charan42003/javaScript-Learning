Document Object Model - DOM

Document Object Model - DOM is an Application Programming Interface(API) for manipulating HTML, XML documents
(adding, removing, & modifying parts of the documents)



Selecting Elements using "id"

        it returns 'null' if ID doesn't exist
        syntax:
            document.getElementById("id_Name");

        example:
            let a = document.getElementById("btn");

Selecting Elements using 'class'

        stores in form of array
        syntax: 
         document.getElementsByClassName("class_name");

        example:
         let a = document.getElementsByClassName("part");
         console.log(a[0]);

Selecting Elements using 'tagName'

        stores in form of array
        syntax:
         document.getElementsByTagName('tag_name');

        example:
            let a = document.getElementsByTagName('p');
            console.log(a[0]);

Selecting Elements using "css Selectors"

  • querySelector - selects only the first element
  • querySelectorAll - selects all the elements with the css property and stores in form of array
        1. querySelector
          syntax:
             document.querySelector('css Selector')

          example:
            let a = document.querySelector('li .menu');
            console.log(a);
        
        2. querySelectorAll
           syntax:
                document.querySelectorAll("css selector");

           example:
                let a = document.querySelectorAll("li");
                console.log(a[2]);

parent and child (indirect selection) 

        1. selecting parent
            syntax: 
                 child_element.parentElement;

        2. selecting first child
            syntax:
                parent_element.firstElementChild;
        
        3. selecting last child
            syntax:
                parent_element.lastElementChild;

        4. selecting all the child elements inside the parent
            syntax;
                parent_element.children;

Sibling selection

        1. previousElementSibling
            syntax: 
                selected_element.previousElementSibling;
                
        2. nextElementSibling
            syntax: 
                selected_element.nextElementSibling;

Mainpulating Elements using 'innerHTML'

        syntax:
            selected_element.innerHTML = " html to Mainpulate";
        
        example:
            let a = document.querySelector(".div");
            a.innerHTML = 'Click me';  // tags can also be given

        .textContent //this will give the inner text of the element

create and append elements

        creating element:
            document.createElement("tag name");

        appending element:
            parent.appendChild("new_created Element");

        example:
            let par = document.getElementById("div");
            let btn = document.createElement("button");
            btn.innerText = "click me";
            btn.className = "hello class";
            par.appendChild(btn);

insertBefore

remove element

       remove element
        syntax:
            parent_Element.removeChild(elementToRemove);

Cloning element

        syntax:
            cloning_Element.cloneNode(true); // true is needed to copy the child elements.

        example:
            cloned = cloning_element.cloneNode(true);
            document.body.appendChild(cloned);

replacing element

        syntax: 
            parent.replaceChild(element_to_replace, target-what to replace);

        example:
            let a = document.getElementById("parent");
            let cr = document.createElement('li');
            cr.innerText = 'newly created';
            let replace = document.querySelector("#child").nextElementSibling;
            a.replaceChild(cr, replace);

changing Attributes

        syntax:
            element.setAttribute('name', "value"); //to set Attributes
            element.removeAttribute('name'); //removes Attribute
            element.getAttribute('name')// to get the value of attribute
            element.hasAttribute('name');// checks if the attribute exists (returns true or false);

changing inline Styling

        syntax:
            element.style.cssText = "properties and values; ....";// this will erase the previous value 
                                                                  // .cssText += "properties and values" 
            (or)
            setAttribute method
            (or)
            element.style.property = "value";

manipulating class

        element.classList; // shows all the classes used
        element.classList = value // adds the  class ;
        element.classList.add('class_name');// adds class
        element.classList.remove('class_name');// removes class
        element.classList.replace("target", "new_class");// replaces the class
        element.classList.contains('class_name');// checks if the class exists returns t/f
        element.classList.toggle('class_name');// toggles the class

Event Bubbling and Event Capturing

event bubbling - child to parent (false //default)
event capturing - parent to child (true) 

        event bubbling
            example:
            let div = document.querySelector("#div");
            let btn = document.querySelector("#button");
            let body = document.body;
            
            body.addEventListener('click', bodyCli, false);//false part could be empty
            div.addEventListener('click', divCli, false);//false part could be empty
            btn.addEventListener('click', btnCli, false);//false part could be empty
            
            function bodyCli() {
                console.log("body clicked");
            }
            function divCli() {
                console.log("div clicked");
            }
            function btnCli() {
                console.log("btn clicked");
            }

        event capturing 
            example:
            let div = document.querySelector("#div");
            let btn = document.querySelector("#button");
            let body = document.body;
            
            body.addEventListener('click', bodyCli, true);
            div.addEventListener('click', divCli, true);
            btn.addEventListener('click', btnCli, true);
            
            function bodyCli() {
                console.log("body clicked");
            }
            function divCli() {
                console.log("div clicked");
            }
            function btnCli() {
                console.log("btn clicked");
            }

Stop Propogation

        it will only trigger the clicked element not along with parent or child or sibling
        // this can be applied with eventBubbling
        example: 
            let div = document.querySelector("#div");
            let btn = document.querySelector("#button");
            let body = document.body;
            
            body.addEventListener('click', bodyCli);
            div.addEventListener('click', divCli);
            btn.addEventListener('click', btnCli);
            
            function bodyCli() {
                console.log("body clicked");
            }
            function divCli() {
                console.log("div clicked");
            }
            function btnCli(event) {
                console.log("btn clicked");
                event.stopPropagation();
            }

Prevent Default

        removes the event assigned to it or
        simply remove the behavour of the element and mainpulate

        example:
            let link = document.querySelector('a');
            link.addEventListener('click', function(e) {
                console.log("clicked");
                e.preventDefault();
            });
