Default parameters

this is to give a default value to the parameter

        syntax: function function_name(parameter1 = value, parametern = value) {}

        example:
            function add(a=2, b=2) {
                console.log(a+b);
            }
            add(); //2 + 2 = 4
            add(5); //5 + 2 = 7
            add(5, 5); //5 + 5 = 7
