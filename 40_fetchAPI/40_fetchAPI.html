Fetch API

fetch API is a two stage process so

        syntax :
        fetch(url)
        .then((value) => {
            return value.json();
        })
        .then((result) => {
            //code result
        })

Post Request

to make a post request. We need to use fetch options

  1. Method -> HHTP, post
  2. body -> the request body
        // POST REQUEST 
        let options = {
            method: "POST",
            headers: {
                "content-type": "application/json"
            },
            body: JSON.stringify({
                title: "foo",
                body: "bar",
                userId: 100,
            })
        }
        fetch('url', options) 
            .then((response) => response.json())
            .then((json) => console.log(json))
