SetInterval & SetTimeout

setTimeout

runs a perticular set of code after the time delay set by the programmer

        syntax:

            setTimeout(function(), timeInMiliSecond);

        example:
            setTimeout(function() {
                alert("hello")
            },1000); // this will alert "hello" after 1sec

setInterval

continuelsy runs a set of code with the time line given

        syntax:
            setInterval(function(), timeInMiliSecond);

        example:
            let i = setInterval(function() {
                console.log(1)
            }, 1000)

        to remove setInterval 
        syntax:
            clearInterval(name of the interval);

        example:
            clearInterval(i) //clearing the previous interval;
